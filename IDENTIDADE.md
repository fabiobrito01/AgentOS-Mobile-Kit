# AgentOS Núcleo

## Marca principal

AgentOS

## Nome da distribuição

AgentOS Núcleo

## Versão

1.0.0

## Idioma oficial

Português do Brasil

## Lema

Organize. Automatize. Evolua.

## Propósito

Transformar o Termux em uma estação de desenvolvimento profissional no Android, com comandos, módulos, banco de dados, plugins, diagnóstico, backup e ferramentas de produtividade.

## Regras de linguagem

- Toda interface deve usar Português do Brasil.
- Todos os menus devem usar Português do Brasil.
- Toda documentação deve usar Português do Brasil.
- Mensagens do sistema devem ser claras e objetivas.
- Termos técnicos podem ser mantidos quando forem nomes próprios ou comandos universais, como Git, GitHub, Termux, Python, Node.js e SQLite.

## Escopo da versão 1.0

- Instalação
- Atualização
- Diagnóstico
- Cópia de segurança
- Restauração
- Banco de dados SQLite
- Gerenciador de plugins
- Explorador GitHub
- Estrutura modular
- Documentação inicial
