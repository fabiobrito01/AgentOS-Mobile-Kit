# Meu celular / Termux

Abra com:

```bash
agentos celular
```

Funcionalidades:

- ver espaco livre;
- listar arquivos grandes dentro de `Download/AgentOS`;
- mostrar caminhos uteis;
- executar limpeza segura;
- corrigir permissao de armazenamento;
- exportar informacoes do ambiente para Download.

Nada nessa tela apaga projetos ou repositorios. A limpeza segura remove apenas temporarios do AgentOS e cache de pacotes do Termux.
