# Assistente de Comandos

Abra com:

```bash
agentos assistente
```

O assistente serve para quem nao quer decorar comando. Ele mostra o comando antes e pergunta se deve executar.

Tarefas atuais:

- instalar ferramentas essenciais;
- autenticar GitHub;
- clonar repositorio por URL;
- atualizar projeto atual;
- enviar projeto atual para GitHub;
- criar chave SSH;
- iniciar sessao tmux.

Isso torna o Termux mais amigavel sem esconder totalmente o que esta acontecendo.
