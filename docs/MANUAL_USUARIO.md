# Manual do Usuario

## Abrir o AgentOS

```bash
agentos
```

## Primeiro uso

1. Entre em `Sistema`.
2. Configure permissoes.
3. Atualize Termux.
4. Configure Git.
5. Faça login no GitHub CLI.
6. Entre em `Configuracoes` e crie as pastas no Download.

## Pesquisar e usar repositorios

1. Abra `GitHub Explorer`.
2. Pesquise uma tecnologia ou ideia.
3. Escolha um resultado.
4. Veja detalhes, clone, fork ou baixe ZIP.

## Trabalhar em projetos

1. Abra `Workspace`.
2. Crie ou selecione um projeto.
3. O AgentOS detecta a tecnologia.
4. Rode comandos sugeridos.
5. Use Git status, pull e push.

## Gerenciar sua conta GitHub

Abra `GitHub Manager` para listar, criar, arquivar ou exportar relatorios.

Exclusao exige confirmacao textual e deve ser usada com cuidado.

## Atualizar o AgentOS

```bash
agentos sistema
```

Escolha `Atualizar AgentOS`.
