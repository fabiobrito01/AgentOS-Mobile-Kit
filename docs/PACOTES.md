# Central de Pacotes

Abra com:

```bash
agentos pacotes
```

Categorias disponiveis:

- Essencial do AgentOS
- Desenvolvimento
- Web e APIs
- Arquivos e produtividade
- Rede e diagnostico
- Banco de dados
- Interface de terminal

Voce tambem pode pesquisar pacotes e instalar manualmente pelo painel.

O painel sempre mostra ou confirma antes de executar instalacoes.
