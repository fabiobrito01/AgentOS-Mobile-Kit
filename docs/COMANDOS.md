# Comandos

## Principal

```bash
agentos
```

Abre a central principal.

## Atualizar Termux

```bash
agentos atualizar
```

Executa atualizacao do Termux e instala as ferramentas essenciais.

## GitHub

```bash
agentos github
```

Abre o menu para pesquisar, clonar e fazer fork de repositorios.

## Projetos

```bash
agentos projetos
```

Abre o menu para gerenciar pastas locais, atualizar repositorios, enviar commits e criar repositorios no GitHub.

## Backup

```bash
agentos backup
```

Cria backup das configuracoes, dados e logs do AgentOS.

## Diagnostico

```bash
agentos doctor
```

Verifica ferramentas, pastas, autenticacao GitHub e rede.

## Configuracoes

```bash
agentos config
```

Permite ver e alterar pastas padrao.
