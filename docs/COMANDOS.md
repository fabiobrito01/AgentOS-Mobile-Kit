# Comandos

## Principal

```bash
agentos
```

Abre a central principal.

## Atualizar Termux

```bash
agentos atualizar
```

Executa atualizacao do Termux e instala as ferramentas essenciais.

## GitHub

```bash
agentos github
```

Abre o menu para pesquisar, clonar e fazer fork de repositorios.

## GitHub Manager

```bash
agentos manager
```

Gerencia repositorios da sua conta: listar, atualizar clones locais, criar, arquivar, excluir com confirmacao e exportar relatorio.

## Auditoria GitHub

```bash
agentos auditoria
```

Lista seus repositorios, classifica, exporta relatorio e abre o Catalogo Turbo de repositorios recomendados.

## Pacotes

```bash
agentos pacotes
```

Abre a central de pacotes por categoria.

## Projetos

```bash
agentos projetos
```

Abre o Workspace inteligente para criar projetos, detectar tecnologias e rodar comandos especificos.

```bash
agentos trabalho
```

Atalho para o mesmo menu.

## Backup

```bash
agentos backup
```

Cria backup das configuracoes, dados e logs do AgentOS.

## Meu celular / Termux

```bash
agentos celular
```

Mostra espaco, caminhos uteis, arquivos grandes, limpeza segura e exportacao de informacoes.

## Sistema

```bash
agentos sistema
```

Atualiza Termux, instala ferramentas, configura permissoes, Git, GitHub CLI, pacotes e atualiza o proprio AgentOS.

## Assistente

```bash
agentos assistente
```

Mostra comandos prontos para tarefas comuns e pergunta antes de executar.

## Diagnostico

```bash
agentos doctor
```

Verifica ferramentas, pastas, autenticacao GitHub e rede.

## Configuracoes

```bash
agentos config
```

Permite ver e alterar pastas padrao.
